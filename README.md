# Fraud Detection for E-commerce and Bank Transactions

## Project Overview
This project focuses on improving fraud detection for both e-commerce transactions and bank credit card transactions. The goal is to build reliable machine learning models that can accurately identify fraudulent activities while minimizing false positives and false negatives.

Fraud detection is a critical problem in financial technology, as undetected fraud leads to financial losses, while incorrectly flagging legitimate transactions harms user experience. This project adopts a data-driven and business-aware approach to address these challenges.

---

## Business Context
The project is developed for **Adey Innovations Inc.**, a financial technology company providing solutions for e-commerce and banking platforms. The company aims to enhance transaction security through advanced analytics, geolocation analysis, and machine learning.

---

## Datasets
The project uses three datasets:

- **Fraud_Data.csv**  
  E-commerce transaction data containing user information, transaction details, device data, and a fraud label.

- **IpAddress_to_Country.csv**  
  Maps IP address ranges to countries, enabling geolocation-based fraud analysis.

- **creditcard.csv**  
  Bank transaction data with anonymized features and highly imbalanced fraud labels.
